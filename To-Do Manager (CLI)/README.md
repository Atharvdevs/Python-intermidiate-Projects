# 📝 To-Do Manager (CLI)

A Python-based command-line application to manage daily tasks efficiently.  
This project demonstrates structured programming, file handling, and CRUD operations using JSON storage.

---

## 🚀 Features

- Add new tasks
- View all tasks
- Remove completed tasks
- Edit existing tasks
- Persistent storage using JSON

---

## 🧠 Concepts Used

- File Handling (JSON read/write)
- Data Structures (Lists)
- Modular Programming (multiple files)
- Error Handling
- Basic CRUD Operations
