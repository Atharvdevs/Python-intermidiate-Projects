🚀 Python Intermediate Projects

This repository contains a collection of intermediate-level Python projects focused on strengthening core programming concepts such as file handling, data structures, modular code design, and problem-solving.

These projects are built to move beyond basic syntax and demonstrate the ability to create functional, structured, and reusable applications using Python.

📂 Projects Included
🔹 Expense Tracker
A file-based application to manage and analyze expenses.
Features:
Add, view, edit, and delete expenses
Category-wise tracking
Total expense calculation
Identify highest spending category
Data stored using JSON

🔹 To-Do Manager
A simple task management system to handle daily tasks efficiently.
Features:
Add and remove tasks
View all tasks
Persistent storage using JSON
Clean and structured code organization

📌 Purpose
The goal of this repository is to:
Practice real-world problem solving
Build structured Python applications
Strengthen intermediate-level programming skills
Prepare for more advanced topics like web development and backend system
