💰 Expense Tracker (CLI)

A Python-based command-line application to manage and analyze personal expenses.
This project demonstrates intermediate-level Python skills including file handling, modular design, and data processing.

🚀 Features

Add new expenses
View all expenses
Edit existing entries
Remove expenses
Calculate total expenses
Identify most used category
Persistent storage using JSON

🧠 Concepts Used

File Handling (JSON read/write)
Data Structures (Lists, Dictionaries)
Modular Programming (multiple files)
Error Handling
Data Aggregation & Analysis

expense-tracker
├── main.py        # Controls program flow
├── input.py       # Handles user input
├── save.py        # Handles data storage
├── view.py        # Displays and processes data
├── data.json      # Stores expense data
